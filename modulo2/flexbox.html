Anotações Flexbox

justify-content (alinhamento horizontal)

Controla como os itens se distribuem no eixo principal (geralmente na horizontal).

ValorDescrição
flex-startItens à esquerda.
flex-endItens à direita.
centerItens centralizados.
space-betweenEspaço igual entre os itens.
space-aroundEspaço igual ao redor dos itens.

align-items (alinhamento vertical)

Define o alinhamento dos itens no eixo cruzado (geralmente vertical).

ValorDescrição
flex-startAlinhado no topo.
flex-endAlinhado na base.
centerAlinhado ao centro vertical.
baselineAlinhado pela linha de base do texto.
stretchItens se esticam para preencher o container.

flex-direction

Define a direção do eixo principal (ordem dos itens).

ValorDescrição
rowHorizontal (esquerda → direita).
row-reverseHorizontal invertido (direita → esquerda).
columnVertical (topo → base).
column-reverseVertical invertido (base → topo).
Obs: Ao usar column, o justify-content passa a atuar na vertical!

order (reordenação)

Permite mudar a ordem de exibição de itens sem alterar o HTML.

Exemplo:

.item { order: 2; }

align-self

Alinha um item individualmente, sobrescrevendo o align-items do container.

Exemplo:

.item { align-self: flex-end; }

flex-wrap (quebra de linha)

Define se os itens quebram para novas linhas.

ValorDescrição
nowrapTodos os itens em uma única linha (padrão).
wrapItens quebram para novas linhas.
wrap-reverseItens quebram em ordem reversa.

flex-flow (atalho)

Combina flex-direction e flex-wrap em uma única linha.

Exemplo:

flex-flow: row wrap;

align-content (espaço entre linhas)

Controla o espaçamento entre as linhas quando há quebra (wrap).

ValorDescrição
flex-startLinhas no topo.
flex-endLinhas na base.
centerLinhas no centro vertical.
space-betweenEspaço entre as linhas.
space-aroundEspaço ao redor das linhas.
stretchLinhas esticadas para ocupar o espaço.
⚠️ Só funciona com múltiplas linhas!
